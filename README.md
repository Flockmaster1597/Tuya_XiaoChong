### 一、方案标题

多功能宠物喂食器

### 二、方案应用场景

地点：家庭

背景：随着目前养宠人士的日渐增多，且多是忙碌的上班族，宠物自动投喂机将成为养宠人士的好帮手，特别是近期疫情流行，宠物自动投喂机可以成为留守宠物的生活保障。

功能：该宠物自动投喂机除自身的一些充电状态获取等基本功能外，还可通过蓝牙、wifi模块连接手机远程操控，可通过手机app编辑喂食计划、获取筒内余粮、控制出食快慢，并可以通过语音提醒🐶进食。配合视频监控工具使用更佳！
